Source Tree is a free to use graphical user interface for Windows and Mac to interact with your Git repositories. It is created and maintained by Atlassian. Source Tree is a good place to start with Git for beginners. It is optional for this course, as RStudio has an integrated GUI for Git as well.

  • Windows
  • MacOS

Install Source Tree by downloading and running this .exe file from Source Tree. Note that if you have separate user and admin accounts, you should run the installers as administrator (right-click on .exe file and select "Run as administrator" instead of double-clicking). Otherwise problems may occur later.

For more detailed installation instructions refer to the documentation by Atlassian Support
